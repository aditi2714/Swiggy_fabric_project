# 🍽️ Swiggy Sales Analytics Dashboard (Microsoft Fabric + Power BI)

## 📌 Overview
This project presents an interactive **Swiggy Sales Analytics Dashboard** built using Microsoft Fabric and Power BI.  

The dashboard provides a comprehensive view of sales performance, customer behavior, and restaurant insights, enabling data-driven decision-making.

## 🎯 Key Insights

### 📈 Monthly Sales Trend
- Shows month-wise revenue performance  
- Identifies peak months (May & August ~ ₹6.79M)  
- Helps understand seasonal demand patterns  

---

### 🍽️ Sales by Food Type
- **Non-Veg dominates (63.7%)**  
- Veg contributes ~36.3%  
- Useful for menu and marketing decisions  

---

### 📅 Daily Trend
- Consistent sales across weekdays  
- Slight variation between weekends and weekdays  

---

### 🏆 Top Restaurants (Top 5)
- KFC leads with highest sales (~₹4.2M)  
- Followed by McDonald's and Pizza Hut  
- Helps identify high-performing partners  

---

### 🗺️ Sales by State
- Karnataka leads (~₹5.5M)  
- Followed by Uttar Pradesh & Telangana  
- Highlights key revenue-generating regions  

---

### 📊 Weekly Trend
- Stable weekly performance  
- Useful for operational planning and forecasting
